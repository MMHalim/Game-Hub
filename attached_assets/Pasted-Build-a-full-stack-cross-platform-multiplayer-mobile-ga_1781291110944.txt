Build a full-stack cross-platform multiplayer mobile game using:

Frontend: Flutter (Dart)
Backend: Supabase (Auth, Database, Realtime, Storage)
Monetization: Subscription + Ads integration
🎮 Game Concept

This is a real-time multiplayer party game where players compete to make others laugh.

Core Gameplay:
Users create or join a game session
Each session has N players (example: 5 players)
Each player enters a 1-minute turn
During their turn:
They must use quotes from a selected celebrity (actor, comedian, singer — Egyptian + international)
The goal is to make others laugh
App measures laughter intensity using microphone input (sound level detection)
Highest laughter score wins the match
🧩 Game Flow
1. Authentication
Email/password login
Facebook login (Meta SDK)
Instagram login (Meta OAuth if possible)
2. Home Screen
Create session
Join session
3. Create Session
Input player names
Select celebrity per player
Generate session ID
4. Gameplay Screen
Turn-based timer system
Real-time microphone audio processing
Live score tracking
Player rotation system
5. Results Screen
Winner announcement
Score breakdown per player
